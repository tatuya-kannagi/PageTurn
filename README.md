# PageTurn

電子書籍リーダーなどを、**端末のハードウェアキーでページ送り**するための Android アプリです。

E-Ink 端末の多くは物理的なページめくりボタンを備えていますが、アプリ側がそのボタンに対応していないと使えません。PageTurn は Android のアクセシビリティサービスとしてキー入力を横取りし、**画面の左右タップ操作に変換**することで、ボタン非対応のアプリでも物理キーでページをめくれるようにします。

---

## 主な機能

- **ハードウェアキー → タップ変換**
  - 左キー → 画面左側（幅 10% / 高さ 50% の位置）をタップ
  - 右キー → 画面右側（幅 90% / 高さ 50% の位置）をタップ
- **押しっぱなしで連続ページ送り** — キーを押し続けると一定間隔で自動的にページが進みます。
- **対象アプリの限定** — 指定したアプリを開いているときだけ動作します。それ以外のアプリの操作には影響しません。
- **キー設定ウィザード** — 「左ページ」「右ページ」のボタンを実際に押すだけで割り当てを登録できます。
- **E-Ink 向け UI** — 高コントラストの白基調デザインで、アニメーションを排除し残像（ゴースト）を抑えています。

---

## 動作環境

- Android 7.0 (API 24) 以上
- 物理キー（ページめくりボタン等）を備えた端末を想定（E-Ink 端末など）

## ダウンロードとインストール（野良 APK）

本アプリは Google Play では配布していません。GitHub の Releases から APK を直接ダウンロードしてインストール（サイドロード）してください。

### 1. APK をダウンロード

このリポジトリの [Releases](../../releases) ページから、最新版の `PageTurn-x.x.apk` を端末にダウンロードします。

### 2. 提供元不明アプリのインストールを許可

野良 APK は既定ではインストールできないため、許可が必要です。

- **Android 8.0 以降**：APK を開こうとすると「この提供元のアプリを許可」する確認が出ます。使用するブラウザやファイルアプリに対して許可してください。
  （`設定` → `アプリ` → 対象アプリ → `提供元不明のアプリ` から事前に許可することもできます）
- **Android 7.x**：`設定` → `セキュリティ` → `提供元不明のアプリ` を有効にします。

### 3. APK をインストール

ダウンロードした APK ファイルをタップし、画面の指示に従ってインストールします。

### 4. アクセシビリティサービスを有効化

インストール後、キー入力を取得するためにアクセシビリティサービスを有効にする必要があります。

1. PageTurn を起動します。
2. 「SERVICE」欄が **無効** の場合、「設定を開く」をタップします。
   （または `設定` → `ユーザー補助（アクセシビリティ）` → `PageTurn`）
3. PageTurn のサービスを **オン** にします。

> **うまく反応しないとき**：インストール直後はアクセシビリティサービスを一度オフ→オンし直すと、キーイベントの取得が有効になる場合があります。

## 使い方

1. **入力キーの指定**
   メイン画面の「KEY BINDINGS」→「入力キーの指定」を開き、案内に従って
   「左ページ移動ボタン」「右ページ移動ボタン」を順に押して登録します。
2. **対象アプリの指定**
   「TARGET APPS」→「アプリの指定」で、ページ送りを有効にしたいアプリを選択して保存します。
3. 指定したアプリを開き、登録したハードウェアキーを押すとページが送られます。

---

## 仕組み

- `PageTurnAccessibilityService` が `onKeyEvent` でハードウェアキーを横取りし、`GestureDescription` でタップジェスチャを発行します。
- 現在最前面のアプリを監視し、対象アプリリストに含まれるときだけ動作します。
- キー設定ウィザードは `MainActivity` でキーイベントを取得して割り当てを記録します。

## ソースからビルドする

```sh
# JDK は Android Studio 同梱の JBR を利用
./gradlew assembleDebug
```

- 主要構成：Kotlin / Jetpack Compose / AccessibilityService
- minSdk 24 / targetSdk 36

ビルドした APK は `app/build/outputs/apk/` 以下に生成されます。
